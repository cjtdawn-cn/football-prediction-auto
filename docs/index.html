预测方法与数据来源

每场比赛的预测结果由以下6个独立组件融合产生，点击比赛卡片中的"预测推理过程"查看各组件的具体贡献。

组件权重方法数据来源
ML集成25%XGBoost + LightGBM + RandomForest 三模型集成xgabora 230K+场历史比赛 (2000-2026, 42个联赛)
市场赔率35%庄家隐含概率反推 + 从Elo推导 (无真实赔率时)Elo概率模型 + 赔率合成算法
Elo评分14%双轨Elo系统: 俱乐部K=32, 国家队K=20-60全历史比赛结果逐场计算
基准模型12%泊松分布 + 联赛平均进球基准各联赛历史进球统计
H2H交锋7%近8场历史对战加权 (近期权重更高)历史比赛记录
